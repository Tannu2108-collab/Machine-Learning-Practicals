# K-Means Clustering Implementation

This project implements the *K-Means Clustering* algorithm, a fundamental unsupervised machine learning technique used to partition unlabeled data into distinct clusters.

## 🛠️ Tech Stack
* *Python*
* *Scikit-Learn*
* *NumPy*

## 📂 Project Overview
K-Means clustering works by grouping $n$ observations into $k$ clusters, where each observation belongs to the cluster with the nearest mean (centroid).


## 🚀 How to Run
1. Install the required dependencies:
   pip install scikit-learn numpy
2. Run the clustering script:
   python kmeans.py

## 📊 Key Concepts
* *Unsupervised Learning:* Learning patterns from data that does not have predefined labels ($y$).
* *Centroids:* The center points that define each cluster.
* *Clustering Logic:* The algorithm iteratively assigns points to the closest centroid and updates the centroid positions until convergence.

---
This implementation demonstrates the ability to extract meaningful groupings from raw, unlabeled datasets.
