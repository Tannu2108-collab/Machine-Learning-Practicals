# Machine Learning Practicals

## 📂 Project Structure
This repository is organized into distinct modules:

1. *[01_Linear_Regression](01_Linear_Regression/):* A basic implementation of simple linear regression to model a direct relationship between input ($X$) and output ($y$).

2. *[02_Multi_Linear_Regression](02_Multi_Linear_Regression/):* Extending regression to handle multiple input features for predicting target values.
   
3. *[03_Unsupervised_Learning](03_Unsupervised_Learning/):* Demonstrating K-Means clustering to partition unlabeled data into distinct groups.
